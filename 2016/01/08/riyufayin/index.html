日语发音及音变规则

  • 日语学习

日语的发音规则

刚开始学日语的时候好多人可能都会有这样的感受：为什么日语的发音这么多这么杂？同一个字的汉字读音为什么会有两三种？常常有种臣妾做不到的无力感。

其实，日语相对来说，在发音方面还是有一些小小的规律可循的。这里为什么要加个“相对”的修饰语呢？因为语言这东西，凡是有规律，就一定存在例外的嘛~~大家一定要理解。想当年学韩语音变的时候，也很能体会到老师在台上的那种无力感了，经常会说：“同学们，这个基本的规律是这样这样的，但是……这个..这个…，嗯，再加这个单词是特殊的，请一定要硬记下来！”

好，今天从两个方面展开来讲一讲日语的“发音规则”和“音变规则”。

发音规则

日本与中国是一衣带水的近邻，想当年我天朝上国国力强盛时，日本还基本上处于未开化状态。而日本人又是一个特别擅长吸收别国先进文化的民族，因此就派遣了大批的人漂洋过海来学习，几乎将中国的文化原封不动的复制了过去，由其是语言文字方面，因此日语中保留了大量的汉字词。

随着我国历史朝代的不断更迭，各民族的不断融合，使得现代汉语与古代的汉语在发音方面有了很大的不同。而日本则由于岛国的独特地理环境，又是一单一的民族，所以没有经受过太多的战乱和民族融合，比较好的保留了古汉语的发音。这样也就造成了现代汉语发音与日语的发音出现了比较大的偏差。

尽管如此，我们的汉语和日语间还是有一定的对应关系的。掌握了这些基本的关系，对日语中汉字词的学习来说，就相当于如虎添翼，事半功倍。

好，闲话少说。

一、单/双音规律

如果单词的汉字在汉语拼音中是单元音（单韵母），一般会发“短音”，如果的汉字在汉语拼音中是双元音（双韵母），一般会发“长音”，这里的长音其实是指双音节了，也就是至少由两个假名拼成。

例如：

字（zi） -> 字（じ）　；　旅（lv） -> 旅（りょ）
主（zhu) -> 主（しゅ） ；　度（du）-> 度（ど）　

招待（zhao dai）-> 招待（しょう　たい）
有料（you liao) -> 有料（ゆう　りょう）

特殊： 开篇时也提到，凡事都有例外，特别是语言方面。所以上面的规律只是在你学到一定程度时，有些单词记忆不是特别清楚，不太知道到底是短音还是长音的时候，利用这种方法正确率会比较高一些而已。

二、拨音规律

汉语的拼音中，一般情况下，韵母以n结尾的音，如an，en,in等，基本上在日语中都会发「ん」的音。

例如：

人（ren）-> 人（にん）；　心（xin） -> 新（しん）
战（zhan）->　戦（せん）
生成（sheng cheng）-> 生成(せい　せい）

特殊： 肯定（こう　てい）

三、长音规律

一般情况下，汉语拼音中的后鼻音ng，在日语中会发成长音。特别是ang，ong，uang,iong这一类，基本上后面都是发[う」长音。

例如：

主张（zhu zhang） -> 主張（しゅ　ちょう）
无常（wu chang） -> 無常（む　じょう）
中国（zhong guo）-> 中国（ちゅう　ごく）
重量（zhong liang） -> 重量（じゅう　りょう）

特别１： 声母为zh,ch,sh时,eng常发「い」音，其他情况多发「う」长音。

例如：

增长（zeng zhang） -> 増長（ぞう　ちょう）
能力（neng li） -> 能力（のう　りょく）
生成（sheng cheng）-> 生成（せい　せい）
整理（zheng li） -> 整理（せい　り）

特别2： 大部分情况下ing常发「い」音。
也有一些情况发「う」长音，发长音的规律性相对来说不像前面的那么有规律，但是也可以特殊记忆一下一些常用的字，比如说发长音的主要有下面一些字： 行（こう）、興（きょう）、幸（こう）、京/境（きょう）、評（ひょう）、庁（ちょう）等。

例如：

英雄（ying xiong） -> 英雄（えい　ゆう）
晴朗（qing lang） -> 晴朗（せい　ろう）
行为（xing wei） -> 行為（こう　い）
经济（jing ji） -> 経済（けい　ざい）

如果掌握了这一些发音规则，作为一名中国人，你会发现你比西洋的那帮老外们在记单词的速度上一定快很多很多。这得感谢我们老祖宗当年强大的文化输出能力，使得我们这些后人能够在学习日语的时候省心不少。

变音规则

说到日语的音变现象，在了解它之前，自己确实迷糊过好长一段时间。明明是同一个字，为什么在这个词里面读成一种音，而到了另外一个词里就读成另外的一种音。当时的感觉就是日语的发音也太不严谨、太随行了吧？后来，直到看了很多的资料介绍以后，才认识到原来日语的很多发音其实都蛮有规律，虽然有变化，但都是按照一种规律来进行的。

那具体的变化如何？一起来看看：

一、促音化

在音读的汉字词中，主要会出现促音化现象，具体如下：

① 前一个字尾音是送气音，后一个头音也是送气音，即当两个送气音相遇时，前一个要变成促音「っ」。

例如：

学（がく）＋校（こう）＝学校（がっこう）
一（いち）＋冊（さつ）＝一冊（いっさつ）
発（はつ）＋達（たつ）＝発達（はったつ）

② 两送气音相遇，若后一个词的头音是「は」行的送气音，前一个变成促音「っ」的同时，后一个的头音要变成它的半浊音。

例如：

発（はつ）＋表（ひょう）＝発表（はっぴょう）
突（とつ）＋破（は）＝突破（とっぱ）
失（しつ）＋敗（はい）＝失敗（しっぱい）

特别：后一个词头音本身是浊音，则合成词不会发生任何音变。
設（せつ）＋備（び）＝設備（せつび）
絶（ぜつ）＋望（ぼう）＝絶望（ぜつぼう）

③ 前一汉字尾音是ん，若后一个词的头音是「は」行的送气音，则变为半浊音，当然，也有极少数是变为浊音。

例如：

心（しん）＋配（はい）＝心配（しんぱい）
神（しん）＋秘（ひ）＝神秘（しんぴ）
何（なん）＋分（ふん）＝何分（なんぷん）

二、浊音化

在训读的时候，也就是日本固有词（和语）构成复合词时，会有浊音化现象。原则上是只有固有词，而汉字词和外来词不会浊化。但请记住我们一直强调的一句话：凡事总有例外！

开头假名为送气音行，即「か」、「さ」、「た」、「は」行的词，接在其他词的后面构成复合词时，送气音假名变为浊音。

例如：

鼻（はな）＋血（ち）＝鼻血（はなぢ）
昔（むかし）＋話（はなし）＝昔話（むかしばなし）
人（ひと）＋人（ひと）＝人々（ひとびと）

例外:

① 本身含有浊音的词，在相连时不发生音变：紙屑（かむくず）、大風（おおかぜ）

② 前一汉字最后一个假名在え段的，变为同一行的あ段假名。

例如：

雨（あめ）　＋　水（みず） → 雨水（あまみず）
稲（いね）　＋　光（ひかり） → 稲光（いなびかり）

日语真的很好玩，到处都是变化。发音会变化、音调会变化、动词和形容词会变化。可能有些人会觉得十分麻烦，十分繁杂。可我们的世界本身就是一个变化的世界啊，我们时时刻刻都在变化，但都有规律可循，这本身难道不是一件让人觉得神奇的事情吗？

自己要加油了，万变不离其宗。我们把变化的东西抽丝剥茧，找到本质、找到规律，就一切尽在我们的掌控当中了。这种掌控的感觉，想想都还觉得兴奋呢。^^
