韩语跟日语的发音对应关系

  • 日语学习
  • 韩语

大家都知道，中日韩同属东亚文化圈，有种说法是中国的很多文化是经过韩国之后，才再传到了日本。像流行的徐福东渡的传说。据说徐福就是首先在韩国的西归浦那里休整了一段时间后，才东渡去的日本。这虽然是一个传说，但也能看出韩国在地缘方面，无形中起到了中日之间纽带的作用。

如果你会韩语，你会发现韩语的发音对应关系比我们现代汉语跟日语的对应关系还要有规则，利用已会的韩语再结合上一篇：日语发音及音变规则刚讲的汉语对应规律会更加提高准确率。

下面就介绍一下韩语跟日语在读音方面的对应关系，与汉语和日语不同，韩语中至今还保留了韵尾，有“入声”和“非入声”两种。现在的粤语至今也保留着相似的入声，如果你懂一点点粤语的话就能感受到了。而日语虽然是开音节语言，但基本上，凡是韩语中有韵尾的汉字，在日语的发音中，基本上都是由两个音节构成。而韩语的韵尾跟日语发音的最后一个音节有着十分规律的对应关系。所以，如果你会韩语，日语一个汉字词音读部分的第二个音节你基本上就能够给推断出来。

具体如何，让我们一起来看一下。

入声韵尾（입성받침）

① “ㄱ”韵尾一般会对应日语的「く」或「き」,一般情况下，如果是あ、う、お段开头的话，结尾会是「く」,如果是い、え段开头动画，结尾会是「き」

例如：

악마 -> 悪魔（あく　ま） -> 恶魔（e mo）；
무역 -> 貿易（ぼう　えき） -> 贸易（mao yi）

② “ㄹ”韵尾一般会对应日语的「つ」或「ち」，同样，一般情况下如果是あ、う、お段开头的话，结尾会是「つ」,如果是い、え段开头动画，结尾会是「ち」

例如：

소설 -> 小説（しょう　せつ）-> 小说（xiao shuo）
자살 -> 自殺（じ　さつ）-> 自杀（zi sha）
칠 -> 七（しち）-> 七（qi）

③ “ㅂ”韵尾一般会对应日语的「う」或「つ」，最早，全部是变成「ふ」，但后来因为发音的分化，逐渐分为两种。至于如何分化的就不再细展开说了。

例如：

갑자 -> 甲子（こう　し）-> 甲子（jia zi）
독립 -> 独立（どく　りつ）-> 独立（du li）

非入声韵尾（비입성받침）

① “ㅇ”韵尾全部都是对应「い」、「う」，根据现在的五十音图标识，如果第一个音节不在「あ」、「い」段，而是以「え」段开始的词，全部发「い」音。而在以前历史上发「あ」段+「う」的字，到现代已经全部都已经转化成为了「お」段+「う」结尾。

例如：

강 -> 江（こう）-> 江（jiang）
생 -> 生（せい）-> 生（sheng）
중 -> 中（ちゅう）-> 中（zhong）

② “ㄴ”韵尾全部对应发拨音「ん」的音。

例如：

신앙 -> 信仰（しん　こう）-> 信仰（xin yang）
인민-> 人民（じん　みん）-> 人民（ren min）
잔업-> 残業（ざん　ぎょう）-> 残业（can ye）

③ “ㅁ”韵尾也是全部对应发拨音「ん」的音，其实在「ん」出现之前，这个韵尾是表示为「む」，来跟上面的“ㄴ”韵尾区分的，而现在已经基本上不做区分了。

例如：

감동 -> 感動（かん　とう）-> 感动（gan dong）
엄격 -> 厳格（げん　かく）-> 严格（yan ge）
심사 -> 審査（しん　さ）-> 审查（shen cha）

特别说明： 两个单词中，前一个词最后一个音节是拨音「ん」，而后面跟着的词第一个音节如果是「あ」行或「や」行时，
1、若对应拨音「ん」的韩文是第②中情况的“ㄴ”韵尾，则后一个词的第一个音节则变为对应「な」行发音。
2、若对应拨音「ん」的韩文是第②中情况的“ㅁ”韵尾，则后一个词的第一个音节则变为对应「ま」行发音。

例如：

천왕사 -> 天王寺（てん のう じ）
음양사 -> 陰陽師（おん　みょう じ)
