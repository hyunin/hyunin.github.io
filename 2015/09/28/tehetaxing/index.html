动词的て形和た形

  • 动词
  • 日语学习

动词的て形和た形也是日语中非常常用的一种变形方法，
其中て形算是动词的另外一种连用形式；用途非常广泛，与韩语的~하고 和 ~해서 很类似；
た形则是则表示过去式，
因为两种变形方法完全一致，所以放到一起来说明。

て形和た形在日语的变形中应该算是最麻烦的一种变化，但其实仍然有规律可循。变形法主要有五段动词（三种变法）、一段动词和カ变和サ变动词三大类。

  • 五段动词

  1. 动词原形词尾以う、つ、る结尾的动词。好多日语初学者一开始学动词的时候好多都是从ます形开始的，若ます前面的假名い、ち、り，则按照如下方法变化：

    把相应的う、つ、る（い、ち、り）变为って或った

    • 買（か）う或（買います）-> 買って、買った
    • 持（も）つ或（持ちます）-> 持って 、持った
    • 乗（の）る或（乗ります）-> 乗って、乗った

  2. 动词原形词尾以く、ぐ结尾的动词。ます形则是ます前面的假名是き、ぎ的则按照如下方法变化：

    把相应的く、ぐ（き、ぎ）变为いて（で）或いた（だ）

    • 書（か）く或（書きます）-> 書いて、書いた
    • 泳（およ）ぐ（泳ぎます）-> 泳いで 、泳いだ

  3. 动词原形词尾以む、ぬ、ぶ结尾的动词。ます形则是ます前面的假名是み、に、び的则按照如下方法变化：

    把相应的む、ぬ、ぶ（み、に、び）变为んで或んだ

    • 飲（の）む或（飲みます）-> 飲んで、飲んだ
    • 死（し）ぬ或（死にます）-> 死んで 、死んだ
    • 遊（あそ）ぶ或（遊びます）-> 遊んで、遊んだ
  • 一段动词
  1. 变形方法相对简单，只需要将动词后面的る去掉，直接添加て或た即可。
    • 認（みと）める -> 認めて、認めた
    • 教（おし）える -> 教えて、教えた
    • 感（かん）じる -> 感じて、感じた
      特殊：行（い）く -> 行って、行った （主要是为了好发音）
  • カ变和サ变动词
  1. カ变变动词：
    • 来(く）る -> 来（き）て、来た
  2. サ变动词：

    把する变成して、した

    • 掃除（そうじ）する -> 掃除して、掃除した
    • 修理（しゅうり）する -> 修理して、修理した
    • 授業（じゅぎょう）する -> 授業して、授業した
